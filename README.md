# PurpleSchool React Native App

A feature-rich Mobile Application built with React Native as part of the advanced mobile development course by PurpleSchool. This application simulates a modern online learning platform, featuring user authentication, server data fetching, course progress tracking, and state management using Jotai.

## ✨ Features

### 🔐 Authentication Flow
* **Sign Up / Sign In:** Secure user onboarding and login screens.
* **Password Recovery:** Complete forgot password flow with email/OTP verification.

### 📚 Course Management
* **Server-Side Data Fetching:** Dynamic loading of available courses, lessons, and categories from the server.
* **Progress Tracking:** Visual indicators showing the percentage of completion for each course.

### 👤 User Profile & Customization
* **Profile Management:** Update personal details, view learning stats, and customize settings.
* **Avatar Upload:** Choose and upload profile pictures directly from the device storage.

### 🧭 Navigation & State
* **Drawer Navigation:** A fluid sidebar menu for quick access to main application sections.
* **Tab & Stack Navigation:** Structured flow between Home, Courses, Profile, and Auth screens.
* **Global State Management:** Powered by **Jotai** for atomic, lightweight, and fast state updates (user session, progress tracking, UI states).

---

## 🛠️ Tech Stack

* **Framework:** React Native (Expo)
* **Language:** TypeScript
* **State Management:** Jotai
* **Navigation:** React Navigation
* **Networking:** Axios / Fetch API
